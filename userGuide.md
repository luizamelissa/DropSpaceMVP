# DropSpace - Guia do Usuário

## Bem-vindo ao DropSpace!

**DropSpace** é sua plataforma completa de dropshipping. Gerencie produtos, lojas, fornecedores e pedidos tudo em um único lugar.

---

## Powered by Manus

**Tech Stack:**
- **Frontend:** React 19 + TypeScript + Tailwind CSS + shadcn/ui
- **Backend:** Express 4 + tRPC 11 + Node.js
- **Database:** MySQL com Drizzle ORM
- **Authentication:** Manus OAuth
- **Deployment:** Auto-scaling infrastructure com global CDN

---

## Usando o DropSpace

### 1. Começando

Ao acessar o site, você verá a página inicial com as principais funcionalidades. Clique em **"Ir para Dashboard"** para começar. Se ainda não tem uma conta, você será redirecionado para fazer login ou cadastro através do Manus OAuth.

### 2. Dashboard

O **Dashboard** é seu centro de controle. Aqui você pode:
- **Visualizar suas lojas** - Veja todas as lojas que você criou
- **Acessar produtos** - Clique em uma loja para gerenciar seus produtos
- **Ver pedidos** - Acompanhe todos os pedidos da sua loja
- **Conectar fornecedores** - Gerencie seus fornecedores

### 3. Gerenciando Produtos

Acesse a seção de **Produtos** para:
- **Adicionar novo produto** - Clique em "Novo Produto" e preencha nome, descrição, preço, custo, estoque, SKU e categoria
- **Editar produto** - Clique em "Editar" no card do produto para modificar suas informações
- **Deletar produto** - Clique em "Deletar" para remover um produto (você será pedido para confirmar)

Cada produto mostra seu preço, estoque disponível e categoria para fácil gerenciamento.

### 4. Gerenciando Fornecedores

Na seção **Fornecedores**, você pode:
- **Adicionar fornecedor** - Clique em "Novo Fornecedor" e preencha nome, email, telefone, website e endereço
- **Visualizar contatos** - Todos os contatos dos fornecedores aparecem em cards para acesso rápido
- **Conectar com fornecedores** - Use os links de email, telefone e website para se comunicar

### 5. Acompanhando Pedidos

A seção **Pedidos** mostra:
- **Lista de pedidos** - Veja todos os pedidos com ID, cliente, total e status
- **Status do pedido** - Acompanhe se está pendente, processando, enviado ou entregue
- **Status de pagamento** - Veja se o pagamento foi realizado ou está pendente
- **Detalhes** - Clique em "Ver" para mais informações sobre cada pedido

### 6. Checkout e Pagamento

Quando um cliente faz uma compra, ele passa por:
1. **Carrinho** - Revisão dos produtos
2. **Entrega** - Preenchimento de dados pessoais e endereço
3. **Pagamento** - Informações do cartão de crédito
4. **Confirmação** - Recebimento do número do pedido

### 7. Perfil e Configurações

Acesse seu **Perfil** para:
- Ver suas informações pessoais
- Data de cadastro
- Tipo de conta

Em **Configurações**, você pode:
- Escolher idioma e moeda
- Gerenciar notificações
- Configurar privacidade
- Sair da sua conta

## Gerenciando Seu Site

Use o **Management UI** para:

- **Settings** - Configure idioma, moeda, fuso horário e notificações
- **Dashboard** - Acompanhe estatísticas e visibilidade do site
- **Database** - Acesse dados diretamente (com SSL habilitado)
- **Secrets** - Gerencie chaves de API e credenciais
